# Backend issue roadmap

This roadmap sequences backend work into stable milestones.

1. Project bootstrap
2. Auth foundation
3. Core finance domain model
4. Free-form transaction ingestion
5. Planned payments and recurring generation
6. Reports foundation (BDR and BDDS)
7. Transaction/account/category CRUD refinement and filtering
8. Credit card lifecycle support
9. Auth hardening and operational polish

Prefer one issue per milestone unless a milestone is too large and needs a follow-up split.
